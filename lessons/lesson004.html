Lesson 004: Local and Remote repos with Git

how to create a local repo and start using git bash to syncronize your local

download gitbash

go to the terminal

cd to change directory
to get to the correct directory

mkdir to make a directory
to make a directory for your repo, e.g. mkdir hatch-up

git init to initialize a local repo
once you are in your directory, git init will cause the directory to remember that git is there

git status
to see if there are any new things to add
add new things to
git add .
to add all things to "staging"

git commit -m "put Message here"
to commit the files to staging (staging is the git repository on your local machine)

git remote add origin "https://github.com/PeterKuperman/hatch-up" - this your github repo URL, not your website

git push origin to push things from your local machine to github
git pull origin to pull things from github to your local machine
