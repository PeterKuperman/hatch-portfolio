Lesson 2: Brainstorm some ideas

With your group (or by yourself) brainstorm 100 ideas for websites -- or at least 20:

Start with Categories:

Accounting, Dentistry, Animation, Games, Art, Painting, Banking,

Then come up with ideas for each category.

Examples:

1. Multiple choice question: A question in text, Give three choices which are images, each with links to a response
[questions & quizzes]

2. Create a button which generates a random background
[random colours]

3. Build a simple painting program
[painting]

4. Build a secret message websites
[secret messages]

5. Build a website that tracks what day it is and gives a message for that day
[calendars]

6. Build a program that paints squares of random colours as you drag your mouse
[painting] [random colours]

7. Change the styling based on clicking, e.g. more drop shadow
[advanced CSS]

8. Build a bouncing ball game
[games] [bouncing balls]

9. Build a cool animation, e.g. when you click on an image, one of 5 animations happen
[animations] [transformations]

10. Build a clock that does something cool (e.g. the hour, minute, second changes the R, G, B of the background)
[clocks] [random colours]

11. Build a dice roller
[games] [probability]

12. Build Rock Paper Scissors
[games]

13. Build an advice giver (e.g. a magic 8 ball)
[questions & quizzes]

14. Build a simulator for bouncing or dropping balls or squares or shapes or images
[animations]

15. Build a painter that paints temporary circles where your mouse is
[painting] [animations]

16. Build a website that gives you info about how how much time has passed since your birthday, and any milestones to celebrate
[calendars]

17. Build a building blocks app
[games] [simulations]

18. Build a quiz about a topic
[questions & quizzes]

19. Build a painter that allows you to smush shapes that randomly appear
[painting] [games]

20. Build a teleprompter app
[real world] [tools]

21. Build a game that involves moving your cursor around
[games] [cursor movement]

22. Build a stopwatch
[clocks] [real world] [tools]
