Part 1:

Go to GitHub.com
pick a username and password
Verify the account

Nagivate to:

https://github.com/dashboard

Part 2:

Click New Repository
For the repository name choose: hatch-portfolio
Check the checkbox that says: Initialize this repository with a README

Part 3:
Click Create New File
name it index.html
put some initial HTML in it:

<!doctype html>
<html>
<head>
<title>Peter's Hatch Portfolio Page</title>
<meta name="description" content="Our first page">
</head>
<body>
This is the first version of the first page of Hatch portfolio. More to be added soon!
</body>
</html>

Go here:

https://ryanstutorials.net/html-tutorial/html-template.php

to learn about what you typed

at the bottom click "Commit New File" (above this button you can put a short and long description for what you are doing, e.g. "loading up my first page")
